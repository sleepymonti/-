# SleepyMonti.by — Система управления доставкой кондитерских изделий

## Требования

- **Visual Studio 2019 / 2022** (Community или выше)
- **.NET Framework 4.7.2** (устанавливается вместе с VS или через Windows Update)
- **SQL Server LocalDB** (входит в Visual Studio, компонент «Data storage and processing»)

## Запуск проекта

1. Откройте файл `SleepyMonti.csproj` в Visual Studio.
2. Нажмите **F5** или «Запуск без отладки» (Ctrl+F5).
3. При первом запуске приложение автоматически:
   - создаст базу данных **SleepyMontiDB** в LocalDB;
   - создаст все необходимые таблицы;
   - заполнит каталог товаров и добавит 3 тестовых курьера.

> Если LocalDB не установлен, скачайте его отдельно:
> https://learn.microsoft.com/en-us/sql/database-engine/configure-windows/sql-server-express-localdb

## Структура проекта

```
SleepyMonti/
├── Program.cs          — точка входа, инициализация БД
├── Models.cs           — классы сущностей (Client, Product, Order, Courier, …)
├── DatabaseHelper.cs   — все операции с SQL Server (ADO.NET)
├── MainForm.cs         — главная форма с 5 вкладками
├── AddClientForm.cs    — диалог добавления клиента
├── AddProductForm.cs   — диалог добавления товара
├── StatusSelectForm.cs — диалог изменения статуса заказа
└── SleepyMonti.csproj  — файл проекта
```

## Функциональность

| Вкладка         | Возможности |
|-----------------|-------------|
| 👥 Клиенты      | Поиск по телефону, добавление, просмотр всех клиентов |
| 🎂 Каталог      | Фильтрация по категории/цене/названию, добавление товара, изменение наличия |
| 📝 Новый заказ  | Выбор клиента, добавление товаров в корзину, автоназначение курьера, оформление |
| 📦 Заказы       | Список всех заказов, фильтр по статусу, изменение статуса |
| 📊 Статистика   | Количество выполненных доставок и активных заказов по каждому курьеру |

## База данных

Таблицы: **Clients**, **Products**, **Couriers**, **Orders**, **OrderItems**

Строка подключения: `(localdb)\MSSQLLocalDB` — изменить при необходимости в файле `DatabaseHelper.cs`.
